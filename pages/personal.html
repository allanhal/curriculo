Sobre mim

Oi, me chamo Allan e sou programador desde 2010. Comecei a programar na faculdade em 2007 e pratiquei bastante na parte de lógica de programação já no começo. Comecei com Java (e um pouco de Pascal) e durante um bom tempo usei apenas essa linguagem. Mais tarde aproveitando a oportunidade para tirar a certificação de OCJP em Java 6. Durante a faculdade fui monitor das disciplinas de programação até que comecei a receber bolsa para ajudar os alunos nessas matérias técnicas. Também fui monitor nos cursos extra-curriculares de Java para Web e JSF.

Como estagiário trabalhei ainda dentro da Unifor na manutenção do site da instituição utilizando iBatis para integração do massivo banco Oracle, Struts para tratamento de requisições e regras de negócio da plataforma, Jquery para manipulações na tela do Portal Unifor.

Ao finalizar o periodo total de 2 anos de estágio comeci a trabalhar na empresa que depois de alguns anos se tornou a GreenMile. Onde o produto principal é um roterizador para frotas de entrega.
Comecei como trabalhando na aplicação em Windows Mobile. Onde foi necessário criar uma nova ferramenta para customização de telas de acordo com cada cliente. Sendo montada uma estrutura para permitir que o cliente montasse a tela que queria ver no mobile. Após um ano e meio entrei para a equipe de desenvolvimento do aplicativo nativo do Android ainda no começo do desenvolvimento do produto, onde passei aproximadamente 1 ano e meio. Com o tempo veio a necessidade de partir para a plataforma nativa Apple onde fui um dos responsáveis pelo início do projeto/arquitetura. Participei também da transição da aplicação nativa para híbrida.Durante todo o meu tempo como desenvolvedor Mobile tive bastante contato com a parte backend. Com isso continuei praticando bastante a parte de Java no backend e aprendi a resolver problemas em toda a stack. Também tive a oportunidade de me desenvolver profissionalmente no atendimento e consultoria após viagens para clientes em vários estados para resolução de pendências e problemas específicos como em Manaus (Coca), Recife (Coca/Solar), São Paulo (Protege) e Rio de Janeiro (White Martins). Esse contato direto com o cliente me tornou alguém que não apenas possuia o conhecimento técnico, mas também conseguia resolvar problemas sem intermediários acelerando bastante o processo onde o programador senta ao lado do cliente e pula etapas de comunicação e ao mesmo tempo alinha o processo interno da empresa. Após 5 anos de excelentes experiências decidi sair para montar minha empresa.

Decidi abrir uma bicicletaria urbana e durante 2 anos aprendi muito sobre empreendedorismo e as dificuldades e prazeres de ter o próprio negócio. A minha sócia hoje é sócia-administrativa e fica responsável pela administração da loja com mais dois funcionários e eu voltei para o mercado de TI.

Atualmente trabalho na GSH (HS) como desenvolvedor full-stack, onde temos uma solução web utilizando VRaptor 3 como legado e uma nova parte já utilizando Backbone com Marionette. No backend utilizamos Spring (MVC, Security) com integração para o banco Oracle via myBatis.

Tenho estudado bibliotecas e frameworks front-end Javascript desde o começo de 2017. Fiz alguns projetos para aprendizado como este currículo e um pequeno gerador de aplicações front-end em Angularjs. Com o estudo do Javascript também veio a curiosidade de usar o NodeJS para criar o servidor utilizando a mesma linguagem do front-end. Como aprendizado criei um servidor REST que venho utilizando como servidor padrão para as minhas aplicações.

Costumo utilizar bicicleta para deslocamentos para o trabalho e coloquei um resumo das minhas atividades aqui ao lado.
